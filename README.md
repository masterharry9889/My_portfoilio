# Aniket's Interactive Portfolio

A visually stunning, 3D interactive portfolio website showcasing projects, skills, and contact information.

## 🚀 Technologies Used
- **Frontend Framework**: React 18
- **Build Tool**: Vite
- **3D Graphics & Animations**: Three.js, React Three Fiber, Framer Motion
- **UI Effects**: ReactBits (Ripple Grid, Magic Bento, Ribbons, Scroll Float)
- **Styling**: Vanilla CSS

## 🛠️ Local Development

1. **Clone the repository**:
   ```bash
   git clone https://github.com/masterharry9889/My_portfoilio.git
   ```

2. **Navigate into the directory**:
   ```bash
   cd My_portfoilio
   ```

3. **Install dependencies**:
   ```bash
   npm install
   ```

4. **Start the development server**:
   ```bash
   npm run dev
   ```
   *The site will be available locally at `http://localhost:5173/`*

## 📦 Build & Deployment

To build the static files for production:
```bash
npm run build
```
This will generate a `dist` folder completely optimized for deployment.

### Recommended Host: Vercel
This project is configured perfectly for zero-configuration Vercel deployment:
1. Go to [Vercel](https://vercel.com/) and click **Add New -> Project**.
2. Select this GitHub repository.
3. Vercel will automatically detect the **Vite** framework.
4. Click **Deploy**. Your site will be live in seconds!
